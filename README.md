# Boostify Growth LTD - Digital Marketing Agency Website & Admin Panel

A comprehensive, full-stack ecommerce and administration platform tailored for a social media marketing agency. This project combines a dynamic, user-facing frontend designed to convert visitors into clients with a sophisticated backend admin panel for managing staff, tracking sales, and publishing content.

🔗 **Live Demo:** [https://boostifygrowth.infy.uk](https://boostifygrowth.infy.uk)

## 🌟 Key Features

### 💻 Client-Facing Frontend
* **Modern & Engaging UI/UX:** Fully responsive design featuring fluid animations, custom loaders, and interactive components designed for maximum user retention.
* **Ecommerce System:** Seamless checkout flow for purchasing monthly social media management subscriptions and individual service packages.
* **User Accounts (Boostify ID):** Secure registration, login, and personalized user dashboards where clients can track their orders, subscriptions, and progress.
* **Boostify Academy (LMS):** Integrated Learning Management System featuring a custom course player, quizzes, and automated certificate generation & validation.
* **Interactive Gamification:** Features like the "Fate Boost Roulette" that encourage users to register and engage with the brand.
* **Boostify AI Assistant:** An integrated custom-built AI chatbot that helps users with personalized social media strategies, trending hashtags, and real-time professional advice.
* **Dynamic Blog & Case Studies:** SEO-optimized blog system and detailed case studies highlighting past client successes.

### 🛡️ Administrative Dashboard
* **Real-time Analytics:** Visual dashboards showcasing monthly sales, subscription revenue, and lifetime earnings using dynamic charts.
* **Staff & Task Management:** Comprehensive system for assigning tasks to staff members, tracking task completion, and monitoring employee attendance/active sessions through interactive heatmaps.
* **Content Management System (CMS):** Admin capabilities to create, edit, and delete blog posts, manage courses, and update site content without touching the codebase.
* **Internal Tools:** Built-in to-do lists and administrative tracking utilities designed to streamline daily agency operations.

## 🛠️ Technology Stack

**Frontend:**
* HTML5 & Vanilla CSS3 (Custom responsive layouts & animations)
* JavaScript (Interactive elements, gamification, and async requests)
* Bootstrap 5 (Grid system and core components)
* Swiper.js (Carousels and sliders)
* ApexCharts (Admin data visualization)

**Backend & Database:**
* PHP (Core backend logic, routing, and session management)
* MySQL (Relational database mapping users, transactions, blogs, and staff tasks)
* Composer (PHP dependency management)

**Architecture Highlights:**
* Secure session-based authentication for both clients and administrative staff.
* Multi-role authorization (Admin vs. Staff access levels).
* Prepared SQL statements to prevent injection vulnerabilities.
* Modular UI components and dynamic data rendering.
